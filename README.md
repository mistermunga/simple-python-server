# Portfolio v2.0

A simple portfolio website containing:

* one HTML page
* stylesheets and a script
* a simple server in vanilla python

## How to use

Before anything, you need `python >3.12`  

Open the folder in a terminal and run:

``` bash
python.exe server.py
```

And that's it. The webpage will begin serving at `localhost:9999`. The app has no dependencies outside vanilla Python.
